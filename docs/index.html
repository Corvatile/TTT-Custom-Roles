Custom Roles for TTT Docs

Welcome to the Custom Role for TTT Docs website! This website is the best place to find information about playing, configuring, and developing with Custom Roles for TTT.

Custom Roles for TTT is an addon for Garry's Mod that adds over 40 new roles to the Trouble in Terrorist Town gamemode, along with a handful of other tweaks and improvements. You can download the addon through the Custom Roles for TTT Steam Workshop page.

If you would like to stay up to date with the development of Custom Roles for TTT, join our Discord server. There you can find announcements for new updates; forums for suggestions, discussion, and support; user-made roles; and you can even join the Custom Roles for TTT beta to get early access to new features. Outside of Discord, you can also keep an eye on our GitHub repository and project board to see planned features and dev builds before they are announced.
